# MegaGroup AI Filler — v0.2

## Что нового

- ✅ **Реальное сохранение работает.** Расширение умеет писать сгенерированный
  Gemini текст обратно в блоки через `POST /-/cms/v2/lp/block/content`.
- ✅ Защита от «выдумок» Gemini — если модель вернёт ключ которого нет в блоке,
  он будет проигнорирован.
- ✅ Защита от потери данных — все служебные поля блока (image, alt, title_type,
  стили) сохраняются как были. Перезаписываются только текстовые.
- ⚠️ **Сухой прогон по-прежнему включён по умолчанию.** Сначала проверь
  что Gemini генерирует адекватный текст, и только потом снимай галочку.

## Установка / обновление

1. Распакуй архив.
2. `chrome://extensions` → найди старую версию → 🔄 «Обновить»,
   или удали и загрузи новую папку.

## Рекомендуемая последовательность работы

1. Открой **тестовую** страницу в конструкторе (не клиентскую!).
2. Открой расширение, заполни Gemini API key, variant_id, ТЗ.
3. **Галочка «Сухой прогон» — включена.**
4. Загрузи блоки → Сгенерировать тексты.
5. Прочитай в логе что нагенерила Gemini. Если нравится — переходи к шагу 6,
   если нет — поправь ТЗ и повтори.
6. Сними галочку «Сухой прогон».
7. Ещё раз нажми «Сгенерировать тексты». Теперь расширение сохранит результат.
8. Обнови страницу редактора (F5) — увидишь новые тексты в блоках.

## Если что-то пошло не так

- В логе будет видна точная ошибка с каждого блока.
- Сохранение делается **поблочно**, так что если один блок упал —
  остальные всё равно сохранятся.
- В конструкторе MegaGroup есть история версий — если совсем плохо,
  можно откатить.
